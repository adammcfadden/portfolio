Huckleberry

A log parsing Ruby gem


GitHub
Huckleberryclose

Huckleberry is a Ruby gem that parses logfiles using regex. Once the gem is installed it can be used as a command line utility, and the parsing criteria can be customized. It can be used locally to analyze logfiles and identify unusual entries, or be set to run at an interval and send an email with a report. Huckleberry is open-source and still under development.

Bit Bizarre

A marketplace using stripe payments


GitHub View Live
Bit Bizarreclose

Bit Bizarre is a digital market place that lets users set up a shop, and shop for items posted by other users. It uses FactoryGirl and Faker gems to seed the database with users and items. AJAX is implimented on the items page to provide a sortable list with pagination. Bit Bizarre uses the Stripe API to process credit card payments. Check out the live site to really see what it can do!
